Tentang Pekerjaan

Selama menjalani program magang sebagai Data Analyst, saya bertanggung jawab membantu berbagai divisi dalam mengolah data operasional menjadi laporan yang mudah dipahami oleh manajemen.

Sebagian besar pekerjaan saya berfokus pada otomatisasi laporan menggunakan Google Sheets, pembuatan dashboard interaktif di Looker Studio, analisis KPI karyawan, validasi data, serta memastikan kualitas data sebelum digunakan sebagai dasar pengambilan keputusan.

Tanggung Jawab

  • Mengumpulkan data operasional dari berbagai divisi perusahaan.
  • Membersihkan (Data Cleaning), memvalidasi, dan memastikan konsistensi data sebelum diproses.
  • Membuat dashboard interaktif menggunakan Google Looker Studio.
  • Mendesain KPI Dashboard untuk monitoring performa karyawan dan divisi.
  • Membuat laporan operasional harian, mingguan, dan bulanan.
  • Mengembangkan otomatisasi menggunakan Google Sheets Formula dan Google Apps Script.
  • Melakukan analisis tren performa berdasarkan data KPI.
  • Berkolaborasi dengan tim HR, Finance, Marketing, dan Management Development Program (MDP).

Project yang Dikerjakan

  • Dashboard KPI Karyawan
  • Dashboard Recruitment
  • Dashboard Pendapatan Per Program
  • Dashboard Finance & Pembayaran
  • Rekonsiliasi Data Peserta
  • Otomatisasi Google Sheets menggunakan Formula ARRAYFORMULA, QUERY, FILTER, LET, SUMIFS, COUNTIFS, IMPORTRANGE, dan Apps Script.
  • Monitoring performa konsultan berdasarkan KPI.
  • Penyusunan dataset untuk kebutuhan dashboard manajemen.

Workflow Pekerjaan

  1. Mengumpulkan data dari berbagai spreadsheet perusahaan.
  2. Melakukan data cleaning, validasi, dan pengecekan duplikasi.
  3. Menyusun formula otomatis agar proses pelaporan lebih efisien.
  4. Mengolah data menjadi dataset siap pakai.
  5. Membuat visualisasi menggunakan Looker Studio.
  6. Menyampaikan insight kepada manajemen.
